# DIY: Engine Air Filter

The M54 uses a panel-type air filter in a plastic airbox.

## Parts Needed
- Air filter element (13 72 1 730 946 or Mann C 25 114/1)

## Procedure

1. Open hood
2. Unclip air filter box lid (4 metal clips)
3. Lift lid and remove old filter
4. Clean airbox interior of debris
5. Install new filter (ensure correct orientation)
6. Close lid and secure all 4 clips

## Upgrade Options
- **K&N drop-in**: Reusable, slightly more flow, re-oil when dirty
- **aFe Pro Dry S**: Reusable, no oil required
- Note: Stock filter flows adequately for stock engines
