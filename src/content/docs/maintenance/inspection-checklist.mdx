# Inspection Checklist

Use this checklist when performing a thorough inspection of your E46.

## Under Hood
- [ ] Engine oil level and condition
- [ ] Coolant level and condition (blue, not rusty)
- [ ] Power steering fluid level (check color: red = ATF, green = CHF 11S)
- [ ] Brake fluid level and color
- [ ] Drive belt condition and tension
- [ ] Intake boot condition (check for cracks)
- [ ] Coolant hoses (squeeze for firmness)
- [ ] Expansion tank condition (look for hairline cracks)
- [ ] Valve cover gasket (oil seepage?)
- [ ] VANOS solenoids (oil leaks)
- [ ] CCV system (check for oil in intake boot)

## Underneath
- [ ] Oil pan gasket (leaks)
- [ ] Power steering rack (leaks)
- [ ] CV boot condition
- [ ] Exhaust hangers and connections
- [ ] Subframe mounting points (cracks!)
- [ ] Control arm bushings
- [ ] Sway bar end links
- [ ] Brake lines and hoses

## Interior / Electrical
- [ ] All window regulators operate
- [ ] Climate control functions
- [ ] All exterior lights working
- [ ] Dashboard warning lights
- [ ] Pixel condition on instrument cluster and radio
- [ ] Door lock actuators
