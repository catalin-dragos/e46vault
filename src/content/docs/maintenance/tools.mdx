# Essential Tools

Having the right tools makes E46 maintenance significantly easier. Here's what you need for common DIY tasks.

## Basic Tool Kit
- Metric socket set (8mm - 22mm)
- Metric wrench set (8mm - 22mm)
- Torx socket set (T20 - T55)
- E-Torx socket set (E8 - E18)
- Allen key set (metric)
- Torque wrench (10-200 Nm range)
- Floor jack and jack stands
- Wheel chocks
- Oil drain pan

## BMW-Specific Tools
- 32mm oil filter cap wrench
- Fan clutch removal tool (32mm thin wrench)
- Coolant system pressure tester
- Camshaft alignment tools (VANOS service)
- Crankshaft pulley holding tool
- Brake piston compression tool (rear - screw type)
- Fuel line disconnect tool

## Diagnostic Tools
- OBD-II scanner (basic code reading)
- BMW-specific: INPA/ISTA with K+DCAN cable
- Bluetooth adapter: BimmerLink/BimmerCode with Vgate iCar2
