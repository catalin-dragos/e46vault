# Fluids Guide

Using the correct fluids is critical for component longevity. The E46 uses specific fluid types that should not be substituted.

## Quick Reference

| System | Specification | Capacity |
|--------|--------------|----------|
| Engine oil | 5W-30 LL-01 | 6.5L |
| Coolant | BMW Blue G48 50/50 | 8.4L |
| Manual trans | MTF-LT-2 | 1.5L |
| Auto trans | ATF LT 71141 | 9.0L total |
| Differential | 75W-90 synthetic | 1.0L |
| Power steering | ATF Dexron III or CHF 11S | 1.0L |
| Brake fluid | DOT 4 | 1.0L |
| Fuel | 91+ AKI premium | 63L |
