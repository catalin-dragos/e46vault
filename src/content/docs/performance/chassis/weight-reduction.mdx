# Weight Reduction

The E46 330 weighs approximately 3,400 lbs (1,540 kg). Every 100 lbs removed improves acceleration, braking, and handling.

## Easy Wins (Street-Friendly)
- Remove spare tire and jack (~40 lbs)
- Lightweight battery (Braille, Antigravity) — save ~25 lbs
- Remove rear seat (track day only) — save ~30 lbs
- Remove trunk carpet and sound deadening — save ~15 lbs
- Lightweight wheels — save 5-8 lbs per corner

## Moderate (May Affect Comfort)
- Remove A/C system — save ~40 lbs
- Delete power steering (manual rack swap) — save ~15 lbs
- Lightweight seats (Sparco, Recaro) — save ~30 lbs per seat
- Remove rear subwoofer and amplifier — save ~15 lbs
- Fiberglass/carbon trunk lid — save ~15 lbs
- Polycarbonate rear window (convertible to coupe) — save ~10 lbs

## Track Build
- Full interior strip — save ~200 lbs total
- Roll bar adds ~40 lbs back
- Realistic track weight: 2,900-3,000 lbs with driver
