# Wheels & Tires Overview

The E46 330 came with several OEM wheel options and supports a wide range of aftermarket fitments.

## OEM Wheel Options

| Style | Size | Bolt Pattern | Offset | Notes |
|-------|------|-------------|--------|-------|
| Style 68 | 16x7 ET47 | 5x120 | 47mm | Base 330i |
| Style 44 | 17x7 ET47 | 5x120 | 47mm | Star spoke |
| Style 135 (ZHP) | 18x8/18x8.5 | 5x120 | 47/50mm | Staggered |
| Style 71 | 17x7 ET47 | 5x120 | 47mm | Cross spoke |

## Bolt Pattern & Hub

- Bolt pattern: 5x120mm
- Hub bore: 72.56mm
- Lug bolt thread: M12x1.5
- Lug bolt torque: 120 Nm
- Lug bolt seat: Conical (cone seat)
