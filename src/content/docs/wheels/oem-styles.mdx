# OEM Wheel Styles

BMW offered numerous wheel styles for the E46 platform throughout its production run.

## Common E46 330 Wheel Styles

### Style 68 (Star Spoke)
- 16x7 ET47
- Standard on many 330i models
- Lightweight, good for winter setup

### Style 44 (5-Spoke Star)
- 17x7 ET47
- Popular sport package wheel

### Style 71 (Cross Spoke)
- 17x7 ET47
- Elegant multi-spoke design

### Style 135 (ZHP Spoke)
- 18x8 front / 18x8.5 rear (staggered)
- ET47 front / ET50 rear
- Exclusive to ZHP Performance Package
- Forged aluminum, lighter than standard

### Style 72 (M Double Spoke)
- 18x8 front / 18x9 rear
- ET47 front / ET42 rear
- E46 M3 wheel, popular upgrade for 330
