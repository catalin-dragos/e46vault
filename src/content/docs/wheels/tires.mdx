# Tire Guide

Choosing the right tires significantly impacts handling, comfort, and safety.

## OEM Tire Sizes

| Wheel Size | Tire Size | Notes |
|-----------|-----------|-------|
| 16x7 | 205/55R16 | Base, good all-season size |
| 17x7 | 225/45R17 | Sport package |
| 18x8/8.5 | 225/40R18 / 255/35R18 | ZHP staggered |

## Recommended Tires

### Summer Performance
- Michelin Pilot Sport 4S
- Continental ExtremeContact Sport 02
- Bridgestone Potenza RE-71RS (track oriented)

### All-Season
- Michelin Pilot Sport All Season 4
- Continental DWS 06 Plus

### Winter
- Bridgestone Blizzak WS90
- Michelin X-Ice Snow
- Use 16" wheels for winter (more sidewall, cheaper tires)
