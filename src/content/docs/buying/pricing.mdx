# Pricing Guide

E46 330 prices vary widely based on condition, mileage, transmission, and options.

## Price Ranges (2024-2025 Market)

| Condition | Price Range | Description |
|-----------|------------|-------------|
| Project/Rough | $2,000-$5,000 | High mileage, deferred maintenance, issues |
| Driver | $5,000-$9,000 | Sorted, regular maintenance, some wear |
| Clean | $9,000-$14,000 | Well-maintained, documented history, clean interior |
| Excellent | $14,000-$20,000 | Low mileage, exceptional condition, ZHP premium |
| Collector (ZHP) | $18,000-$30,000+ | Low mileage, pristine, full documentation |

## Value Factors

### Adds Value
- Manual transmission (+$2,000-$3,000 over auto)
- ZHP Performance Package (+$5,000-$10,000)
- Low mileage (under 100k)
- Complete service records
- Recently refreshed cooling system
- M-Sport appearance package
- Rare colors (Topaz Blue, Phoenix Yellow)
- No accidents or rust

### Reduces Value
- Automatic transmission
- Missing service records
- High mileage (over 150k)
- Accident history
- Rust (especially structural)
- Deferred maintenance
- Modified with cheap parts
- Salvage title
