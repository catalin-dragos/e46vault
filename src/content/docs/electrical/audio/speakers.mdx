## OEM Speaker Locations and Sizes

### Standard Audio
| Location | Size | Type |
|----------|------|------|
| Front doors (lower) | 6.5 in / 165mm | Woofer |
| Front doors (upper) | 1 in / 25mm | Tweeter |
| Rear deck (sedan) / rear panel (coupe) | 6.5 in / 165mm | Full-range |

### HiFi System (S676A)
| Location | Size | Type |
|----------|------|------|
| Front doors (lower) | 6.5 in / 165mm | Woofer |
| Front doors (upper) | 1 in / 25mm | Tweeter |
| Rear deck / panel | 6.5 in / 165mm | Woofer |
| Rear shelf (sedan) | 1 in / 25mm | Tweeter |
| Under front seats | 8 in / 200mm | Subwoofer (2x) |

## Upgrade Path

1. Replace speakers with quality aftermarket (Focal, Hertz, Morel)
2. Add a dedicated amplifier
3. Add a subwoofer (stealth enclosure in trunk or under-seat)
4. Sound deadening (doors, trunk) for reduced road noise
