The General Module (GM, also called ZKE) is the body control module that manages comfort and convenience functions.

## Functions

- Central locking (doors, trunk)
- Power window control
- Power mirror control
- Interior lighting (dome, footwell, vanity)
- Remote key fob functions
- Crash signal (unlocks doors on airbag deployment)
- Anti-theft alarm system
- Comfort features (auto-lock, selective unlocking)

## Location

Behind the glove box area, mounted to the body.

## Common Issues

- Window regulator control failure
- Central locking intermittent operation
- Interior light timer malfunction
- Comfort coding changes after battery replacement (module may need re-coding)

## Coding

The GM can be coded to customize comfort features (auto-lock speed, selective unlocking, etc.) using BMW diagnostic tools or aftermarket coding software (NCS Expert, Bimmercode).
