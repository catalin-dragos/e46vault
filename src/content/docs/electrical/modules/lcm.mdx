The LCM (Licht Control Modul) manages all exterior lighting functions and monitors bulb status.

## Functions

- Headlight control (low beam, high beam, flash-to-pass)
- Tail light and brake light control
- Turn signal control
- Fog light control
- Daytime running lights (if coded)
- Bulb monitoring and failure warnings
- Automatic headlights (with RLS sensor)
- Welcome lights (coded feature)

## Location

Driver's side footwell, behind the left kick panel.

## Common Issues

- Bulb failure warning when no bulb is blown (module sensitivity)
- Intermittent lighting issues
- Module failure after water intrusion (driver's footwell leaks)

## Coding Options

The LCM can be coded for:
- Daytime running lights (DRL)
- US/Euro lighting spec
- Welcome lights (headlights flash on unlock)
- Corner light behavior
