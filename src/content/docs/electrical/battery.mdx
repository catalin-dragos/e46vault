The E46 battery is located in the trunk on the right side, under a cover panel.

## Specifications

| Parameter | Value |
|-----------|-------|
| Type | Lead-acid (AGM optional) |
| Voltage | 12V |
| Capacity | 80Ah (standard) |
| CCA | 640A (minimum) |
| Group size | 48/H6 |
| Terminal | Top-post |

## Trunk-Mounted Battery

Benefits of trunk-mounted battery:
- Better weight distribution (front-rear balance)
- Protected from engine bay heat

The engine bay has a positive terminal jump start post on the right strut tower area.

## Replacement Notes

- Disconnect negative terminal first, reconnect last
- After battery replacement, some modules may need re-coding (radio, windows)
- Register the new battery with BMW diagnostic software if using AGM type
- Perform the window reset procedure after reconnecting: hold each window switch up for 5 seconds after window is fully closed
