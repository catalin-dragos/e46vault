# ZHP Performance Package

The ZHP (Zentrale Handling Paket) was a dealer-installed performance package available exclusively in North America from 2003-2006. It represents the pinnacle of the E46 330 platform.

## ZHP Enhancements

### Engine
- Revised DME software: 235 hp (+10), 222 lb-ft (+8)
- Higher rev limit: 6,800 RPM
- Quicker throttle response
- Performance exhaust with quad tips (sedan) or dual tips (coupe)

### Transmission
- Close-ratio 6-speed manual (Getrag S6S 420G) â€” ZHP only
- Short shift kit from factory
- 3.07:1 final drive ratio

### Suspension
- M Sport suspension (firmer springs and dampers)
- Sport-tuned steering
- 18" Style 135 forged wheels (8" front / 8.5" rear)

### Exterior
- M-Tech II front bumper, rear bumper, and side skirts
- Quad exhaust tips (sedan)
- Alcantara interior trim
- Unique gauge cluster with red needles
- Illuminated shift knob

## Production Numbers
- Total ZHP production: ~8,600 units (N. America)
- Available: sedan and coupe
- Years: 2003-2006

## Identification
- VIN decode shows ZPP (Zentrale Handling Paket) option
- M-Tech II body kit
- 18" Style 135 wheels
- Red gauge needles
- 6-speed manual only (no auto ZHP)
