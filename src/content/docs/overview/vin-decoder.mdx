# VIN Decoder

Every BMW E46 has a 17-character Vehicle Identification Number (VIN) that encodes production details, options, and specifications.

## VIN Location
- Driver's door jamb sticker
- Dashboard (visible through windshield, driver side)
- Engine bay (stamped on front strut tower)

## VIN Structure

| Position | Meaning | Example |
|----------|---------|---------|
| 1-3 | World Manufacturer ID | WBA = BMW AG |
| 4-5 | Model series | EV = E46 330Ci |
| 6-7 | Body type / engine | 33 = 330i sedan |
| 8 | Safety/emissions | 5 = US market |
| 9 | Check digit | Calculated |
| 10 | Model year | 3 = 2003 |
| 11 | Assembly plant | A = Munich, F = Regensburg |
| 12-17 | Sequential number | Production sequence |

## Common E46 330 VIN Prefixes
- **WBAEV334**: 330Ci coupe
- **WBAEW334**: 330i sedan
- **WBAEW534**: 330xi sedan (AWD)
