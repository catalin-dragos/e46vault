# Paint Codes

Your paint code is located on a sticker in the driver's door jamb or engine bay. It's needed for touch-up paint and body work.

## Finding Your Paint Code
1. Open driver's door
2. Look at the label on the door jamb or B-pillar
3. Paint code is typically a 3-digit number
4. Use this code when ordering touch-up paint or matching for body work
