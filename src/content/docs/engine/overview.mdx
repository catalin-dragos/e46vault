# Engine Overview — M54B30

The M54B30 is a 3.0-liter naturally aspirated inline-6 producing 225 hp and 214 lb-ft of torque. It's widely regarded as one of BMW's best engines.

## Key Specifications

| Specification | Value |
|--------------|-------|
| Code | M54B30 |
| Configuration | Inline-6, DOHC, 24-valve |
| Displacement | 2,979 cc |
| Bore x Stroke | 84.0 x 89.6 mm |
| Compression | 10.2:1 |
| Power | 225 hp @ 5,900 RPM |
| Torque | 214 lb-ft @ 3,500 RPM |
| Redline | 6,500 RPM |
| Variable valve timing | Dual VANOS (intake + exhaust) |
| Engine management | Siemens MS 43 DME |
| Fuel requirement | Premium 91+ AKI |
| Oil capacity | 6.5 liters with filter |

## Engine Systems

- **VANOS**: Variable valve timing for both intake and exhaust cams
- **DISA**: Variable-length intake manifold for broad torque curve
- **EWS**: Electronic immobilizer integrated with DME
- **CCV**: Crankcase ventilation system (PCV equivalent)
- **Secondary air**: Emissions system for cold-start catalyst heating

## Reliability

The M54 is exceptionally reliable when properly maintained. The most critical maintenance item is the cooling system — see the Cooling section for details.
