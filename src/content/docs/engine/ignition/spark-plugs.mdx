# Spark Plugs

The M54 uses 6 spark plugs, one per cylinder. OE specification is Bosch FR7LPP332W (platinum) or NGK BKR6EQUP.

## Specifications
- Thread: M12 x 1.25
- Reach: 26.5mm
- Gap: 0.7-0.8mm (pre-gapped)
- Torque: 25 Nm
- Type: Single platinum (OE)

## Replacement Interval
45,000-60,000 miles. See DIY section for procedure.

## Upgrade Options
- Iridium plugs (NGK BKR6EIX): Longer life, sharper spark
- OE platinum: Best value for stock engines
