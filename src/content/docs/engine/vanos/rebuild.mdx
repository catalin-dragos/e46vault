# VANOS Rebuild

Replacing the VANOS seals is the most common repair for VANOS issues. The Beisan Systems seal kit is the most popular solution.

## Parts Needed
- Beisan Systems VANOS seal kit (or equivalent)
- VANOS solenoid O-rings
- Camshaft alignment tools
- New VANOS unit bolts (recommended)

## Procedure Overview
1. Remove valve cover
2. Set engine to TDC cylinder 1
3. Install camshaft alignment tools
4. Remove VANOS unit (6 bolts)
5. Disassemble VANOS unit
6. Replace Teflon piston seals
7. Replace solenoid O-rings
8. Reassemble VANOS unit
9. Reinstall on cylinder head
10. Remove alignment tools
11. Reinstall valve cover
12. Clear fault codes and test drive

## Critical Notes
- Engine MUST be at TDC with alignment tools installed
- Do not rotate engine with VANOS removed
- Mark timing chain position before removal
- Use new bolts for VANOS unit installation (10 Nm)
