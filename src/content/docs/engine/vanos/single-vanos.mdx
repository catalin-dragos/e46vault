# Single vs Double VANOS

## Comparison

| Feature | Single VANOS | Double VANOS (M54) |
|---------|-------------|-------------------|
| Camshafts adjusted | Intake only | Intake + exhaust |
| Adjustment type | Continuous | Continuous |
| Found on | M50TU, M52, S50/S52 | M54, M56, S54 |
| Seal type | Different design | Teflon piston seals |

## Impact on M54 Performance
Double VANOS provides:
- 10-15% broader torque curve
- Improved emissions compliance
- Better idle quality
- The characteristic smooth BMW inline-6 feel

## Identifying Your System
All E46 330 models (1999-2006) use double VANOS. If you have an M54B30, you have double VANOS.
