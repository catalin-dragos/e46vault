# Seats

## Seat Types
- Standard seats (manual adjust)
- Power seats (with or without memory)
- Sport seats (more bolstering, manual or power)
- Heated seats (option S494A)

## Common Issues
- Seat heater element failure
- Power seat motor failure
- Memory module issues
- Leather cracking and wear
- Sport seat side bolster wear (driver side)
- Seat tilt mechanism breaking

## Seat Covers / Materials
- Leatherette (vinyl, base models)
- Leather (most 330 models)
- Alcantara/leather (ZHP, M-Sport)

## Aftermarket Upgrades
- Recaro seats (track use)
- Sparco seats (track use)
- M3 seat swap (direct fit with correct brackets)
