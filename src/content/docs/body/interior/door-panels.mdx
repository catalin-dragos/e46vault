# Door Panels

## Removal Procedure
1. Remove screw behind door handle pull
2. Pry out window switch panel
3. Disconnect window switch connector
4. Remove any additional visible screws
5. Use trim removal tool to pop retaining clips around perimeter
6. Lift panel up and off door
7. Disconnect any remaining connectors (speaker, light)

## Common Issues
- Retaining clips breaking (have spares ready)
- Door pocket sagging
- Speaker grille damage
- Leather/vinyl peeling on grab handle
- Interior door handle cable disconnecting
