# Key Fob

## Battery Replacement
1. Pry open key fob shell (small slot on side)
2. Replace CR2032 battery
3. Reassemble shell
4. No reprogramming needed after battery replacement

## Key Fob Re-sync
If fob stops working after battery change:
1. Stand next to car with key
2. Press unlock on fob
3. Insert key into driver's door lock
4. Turn key to unlock position 3 times within 10 seconds
5. Fob should re-sync

## Spare Keys
- BMW dealer can program new keys (~$200-$400)
- Requires EWS (Electronic Wegfahrsperre) programming
- Maximum of 10 keys can be programmed to one car
