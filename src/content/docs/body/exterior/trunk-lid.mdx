# Trunk Lid

## Trunk Release
- Key lock cylinder
- Interior release button (center console)
- Key fob remote

## Common Issues
- Trunk lid struts failing (trunk doesn't stay open)
- Lock actuator failure
- License plate light bulb burnout
- Water leak from gasket deterioration
- Trunk hinge area rust

## Trunk Strut Replacement
1. Support trunk lid in open position
2. Pop old strut ball sockets off
3. Press new struts onto ball studs
4. Close and test
