# Brakes Overview

The BMW E46 330 features a robust braking system with ventilated discs front and rear. The 330 models received larger brakes than the 325, making them popular for swap upgrades.

## Brake Specifications

| Component | Front | Rear |
|-----------|-------|------|
| Rotor diameter | 325mm | 320mm |
| Rotor thickness (new) | 25mm | 20mm |
| Rotor minimum thickness | 22.4mm | 17.4mm |
| Caliper type | Single-piston floating | Single-piston floating |
| Pad compound (OE) | Semi-metallic | Semi-metallic |

## Brake System Components

- Ventilated disc rotors (front and rear)
- Single-piston floating calipers
- Hydraulic brake booster
- ABS (Anti-lock Braking System)
- DSC (Dynamic Stability Control) on most models
- Parking brake (drum-in-disc rear)
