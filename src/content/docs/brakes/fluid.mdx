# Brake Fluid

Brake fluid is hygroscopic (absorbs moisture) and must be replaced regularly for safe braking performance.

## Specifications

- **Type**: DOT 4
- **Recommended**: ATE Typ 200 (gold/blue) or Motul RBF600
- **Capacity**: ~1.0 liter for complete flush
- **Interval**: Every 2 years regardless of mileage

## Flush / Bleeding Procedure

Bleed in this order (furthest from master cylinder first):
1. Right rear
2. Left rear
3. Right front
4. Left front

### Two-Person Method
1. One person pumps brake pedal and holds
2. Second person opens bleeder valve
3. Close valve before pedal reaches floor
4. Repeat until clean fluid flows (no bubbles)

### Pressure Bleeder Method
1. Attach pressure bleeder to reservoir (15-20 PSI)
2. Open each bleeder in sequence
3. Flow ~250ml through each caliper
4. Close bleeder, check pedal feel

## ABS Bleeding

After air enters the ABS module, a scan tool bleed cycle may be required:
- Use INPA, ISTA, or compatible tool
- Run ABS pump bleed function
- Then perform standard bleeding procedure
