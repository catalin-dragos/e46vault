# Front Brakes

The E46 330 front brakes use 325x25mm ventilated rotors with single-piston floating calipers.

## Pad Replacement Procedure

1. Raise vehicle, remove wheel
2. Remove caliper guide pin bolts (2x, 30 Nm)
3. Swing caliper up and support with wire (do not hang by brake line)
4. Remove old pads and anti-rattle clips
5. Compress piston with C-clamp (open bleeder to protect ABS)
6. Install new pads and clips
7. Re-install caliper, torque guide pins to 30 Nm
8. Pump brake pedal before driving!

## Rotor Replacement

1. Remove caliper (as above)
2. Remove caliper carrier bolts (2x, 110 Nm)
3. Remove set screw holding rotor to hub
4. Pull rotor off hub (may need persuasion)
5. Clean hub surface
6. Install new rotor, set screw to 6 Nm
7. Reassemble caliper carrier (110 Nm) and caliper (30 Nm)

## Recommended Parts

- **Rotors**: Zimmermann, Brembo, or ATE
- **Pads**: Akebono Euro, Hawk HPS, or StopTech Street
