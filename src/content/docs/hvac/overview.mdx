# HVAC Overview

The E46 uses the IHKA (Integrierte Heizung Klima Automatik) automatic climate control system on most 330 models.

## System Components
- IHKA control panel (center console)
- Blower motor with final stage resistor (FSR)
- Heater core (uses engine coolant)
- A/C compressor (R134a)
- Evaporator core
- Blend doors (temperature and direction)
- Cabin air filters (2x microfilters)
- A/C condenser (in front of radiator)

## Common Issues (Quick Reference)
- Blower motor FSR failure (most common)
- Blend door actuator failure
- Heater core leak
- A/C compressor failure
- Cabin filter neglect
