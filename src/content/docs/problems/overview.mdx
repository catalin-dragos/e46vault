# Common Problems Overview

The E46 330 is a well-engineered car, but age and mileage bring predictable failures. This section documents the most common issues, their symptoms, and solutions.

## Critical / Safety Issues
- **Cooling system failure** — can cause catastrophic engine damage
- **Rear subframe cracking** — structural integrity concern
- **Control arm failure** — affects steering and handling
- **Brake wear sensor** — early warning of pad wear

## Very Common Issues
- Window regulator failure
- VANOS seal degradation
- Expansion tank cracking
- Blower motor final stage resistor (FSR) failure
- DISA valve flap failure
- CCV (crankcase ventilation) valve failure
- Ignition coil failure

## Common Age-Related Issues
- Cooling system hose degradation
- Bushing wear (control arms, RTABs)
- Door lock actuator failure
- Instrument cluster pixel loss
- Convertible top hydraulic issues (330Ci)
