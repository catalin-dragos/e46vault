# Front Sway Bar

The front sway bar (stabilizer bar) connects both sides of the front suspension to reduce body roll in corners.

## Specifications

- Stock diameter: 24.5mm solid
- Material: Spring steel
- Connected via end links to strut body

## Common Issues

- **Worn end links**: Most common failure â€” causes clunking over bumps
- **Worn bushings**: Rubber bushings wear, causing looseness
- **Bent bar**: Rare, usually from impact damage

## End Link Replacement

End links are a common wear item and easy to replace:
1. Raise vehicle and remove wheel
2. Remove upper nut (hold link body with wrench)
3. Remove lower nut
4. Install new end link
5. Torque end link nuts: 59 Nm, bracket bolts: 22 Nm

## Upgrade Options

- **H&R front sway bar**: 28mm, significantly reduces body roll
- **Whiteline adjustable**: Multiple stiffness settings
- **Turner Motorsport**: Various sizes available
