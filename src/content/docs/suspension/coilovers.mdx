# Coilover Guide

Coilover suspension systems replace the factory springs and shocks with adjustable, integrated units. This guide covers selection, installation, and tuning for the E46 platform.

## Popular Coilover Options

### Budget Tier
- **BC Racing BR Series**: Good value, 30-level damping adjustment
- **Raceland Ultimo**: Basic adjustable, limited dampening control

### Mid-Range
- **Bilstein B14/B16**: OEM-quality ride with adjustability
- **KW V1/V2/V3**: German-engineered, excellent street comfort
- **ST Suspensions (by KW)**: Budget KW alternative

## Installation Considerations

- Spring compressor required for stock spring removal
- Alignment mandatory after installation
- Torque strut top nuts to 64 Nm
- Set initial ride height before alignment
- Allow 500 miles for settling before final alignment
