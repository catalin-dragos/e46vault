# Suspension Overview

The BMW E46 uses a MacPherson strut front suspension and multi-link rear suspension. This section covers all suspension components, upgrades, and alignment specifications for the 330i, 330Ci, and 330xi.

## Front Suspension Design

The front suspension is a MacPherson strut design with:
- Aluminum lower control arms with ball joints
- Coil springs with separate shock absorbers
- Stabilizer bar with end links
- Strut mounts with integrated bearings

## Rear Suspension Design

The rear uses a sophisticated 5-link (multi-link) independent design with:
- Central arm (trailing arm)
- Upper control arm
- Lower control arm
- Toe link (integral link)
- Camber link

## Key Specifications

| Parameter | Front | Rear |
|-----------|-------|------|
| Type | MacPherson strut | Multi-link |
| Spring rate (stock) | ~30 N/mm | ~55 N/mm |
| Camber (stock) | -0°30' ± 30' | -1°50' ± 20' |
| Toe (stock) | +0°05' ± 5' | +0°10' ± 10' |
| Caster | +7°30' ± 30' | N/A |

## Ride Height

Stock ride height from center of wheel to fender lip:
- Front: approximately 370mm (14.6")
- Rear: approximately 375mm (14.8")
