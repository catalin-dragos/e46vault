The Getrag S6S-420G is a 5-speed manual transmission used in the E46 330i and 330Ci. It is a robust unit shared with the E39 530i.

## Specifications

| Parameter | Value |
|-----------|-------|
| Manufacturer | Getrag |
| Model | S6S-420G |
| Gears | 5 forward + reverse |
| Fluid type | BMW MTF-LT-2 (75W-80) |
| Fluid capacity | 1.0L |
| Weight | ~35 kg |

## Gear Ratios

| Gear | Ratio |
|------|-------|
| 1st | 4.21 |
| 2nd | 2.49 |
| 3rd | 1.66 |
| 4th | 1.24 |
| 5th | 1.00 |
| Reverse | 3.85 |

## Reliability

The Getrag S6S-420G is an extremely durable transmission that handles the M54 torque well. Common issues are limited to shifter bushings wearing (resulting in sloppy shift feel) and occasional input shaft bearing noise at very high mileage.
