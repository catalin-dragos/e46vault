The E46 330 uses a rear-mounted differential housed in an aluminum case.

## Specifications

| Configuration | Ratio | Type |
|--------------|-------|------|
| 330i/Ci Manual | 3.07:1 | Open |
| 330i/Ci Automatic | 3.46:1 | Open |
| 330xi (all) | 3.46:1 | Open |
| M3 | 3.62:1 | LSD (Type 210) |

## Differential Types

- **Open differential**: Standard on all 330 models
- **Limited-slip differential (LSD)**: Available as aftermarket upgrade

## Identification

The ratio is stamped on a metal tag on the housing. Cover bolt pattern identifies type (small=168, large=188/210).
