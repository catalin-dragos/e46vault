The clutch system on the manual E46 330 consists of the clutch disc, pressure plate, release bearing, pilot bearing, dual-mass flywheel, and hydraulic actuation system.

## Components

| Component | Description |
|-----------|------------|
| Clutch disc | Friction disc that transmits torque |
| Pressure plate | Spring-loaded plate that clamps disc to flywheel |
| Release bearing | Actuates pressure plate diaphragm spring |
| Pilot bearing | Centers input shaft in crankshaft |
| Dual-mass flywheel | Dampens drivetrain vibration |
| Clutch master cylinder | Converts pedal input to hydraulic pressure |
| Clutch slave cylinder | Actuates release fork/bearing |

## OEM Supplier
