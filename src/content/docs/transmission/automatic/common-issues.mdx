## Common Issues

### Harsh Shifting
- Cause: Old ATF, adaptation values need reset
- Fix: Fluid change, adaptation reset with diagnostic software

### Slipping
- Cause: Low fluid, worn clutch packs
Fix: Check fluid level

### Limp Mode
- Cause: Electrical fault, solenoid failure
- Symptoms: Stuck in 3rd gear
- Fix: Scan for fault codes

### Delayed Engagement
- Cause: Low fluid, worn seals
- Fix: Check fluid level

### Torque Converter Shudder
- Cause: Worn lockup clutch
- Symptoms: Vibration at highway speeds
- Fix: Torque converter replacement or rebuild
