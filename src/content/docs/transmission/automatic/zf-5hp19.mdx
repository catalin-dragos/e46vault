The ZF 5HP19 is a 5-speed automatic used in later E46 330 models and the E39 5 Series.

## Specifications

| Parameter | Value |
|-----------|-------|
| Manufacturer | ZF Friedrichshafen |
| Model | 5HP19 |
| Gears | 5 forward + reverse |
| Fluid type | Esso LT 71141 / Lifeguard 6 |
| Pan capacity | ~4.0L (drain and fill) |
| Total capacity | ~9.0L |

## Identification

Identify by transmission oil pan shape:
- **GM A5S 390R**: Oval-shaped pan
- **ZF 5HP19**: Rectangular pan with rounded corners

## Reliability

The ZF 5HP19 is well-regarded. Main issues involve the mechatronic sleeve connector seal, which can leak and cause erratic shifting.
