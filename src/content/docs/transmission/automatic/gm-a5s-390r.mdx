The GM A5S 390R is a 5-speed automatic transmission used in earlier E46 330 models.

## Specifications

| Parameter | Value |
|-----------|-------|
| Manufacturer | General Motors (Strasbourg) |
| Model | A5S 390R |
| Gears | 5 forward + reverse |
| Fluid type | Esso LT 71141 / BMW ATF-2 |
| Pan capacity | ~4.0L (drain and fill) |
| Total capacity | ~9.0L |

## Features

- Adaptive shift programming (learns driving style)
- Sport mode (S) for later upshifts and earlier downshifts
- Steptronic manual mode
- Torque converter lockup in 4th and 5th gear

## Reliability

The A5S 390R is generally reliable with proper fluid maintenance. Regular fluid changes every 60,000 miles are essential for longevity.
