# Fault Code Reference

Common diagnostic trouble codes (DTCs) for the BMW E46 330. Read codes with any OBD-II scanner; BMW-specific codes require INPA, ISTA, or compatible software.

## VANOS Related
| Code | Description | Common Causes |
|------|-------------|---------------|
| P0010 | Intake cam position actuator circuit | Failed solenoid, wiring, low oil |
| P0011 | Intake cam timing over-advanced | Worn VANOS seals, clogged oil passages |
| P0012 | Intake cam timing over-retarded | Worn seals, failed solenoid, low oil |
| P0015 | Exhaust cam timing over-retarded | Worn seals, failed exhaust solenoid |
| P1519 | Intake cam control malfunction | VANOS unit failure, oil supply issue |

## Ignition / Misfire
| Code | Description | Common Causes |
|------|-------------|---------------|
| P0300 | Random/multiple cylinder misfire | Spark plugs, coils, vacuum leak |
| P0301-P0306 | Cylinder 1-6 misfire | Failed coil, worn plug, injector |

## Fuel System
| Code | Description | Common Causes |
|------|-------------|---------------|
| P0171 | System too lean (Bank 1) | Vacuum leak, MAF, intake boot |
| P0174 | System too lean (Bank 2) | Vacuum leak, CCV, DISA |
| P1523 | DISA valve malfunction | Broken flap pin, failed motor |

## Cooling
| Code | Description | Common Causes |
|------|-------------|---------------|
| P0128 | Coolant below regulating temp | Stuck-open thermostat, temp sensor |
