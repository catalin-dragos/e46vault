# Abbreviations & Glossary

## BMW-Specific

| Abbreviation | Full Name | Description |
|-------------|-----------|-------------|
| ABS | Anti-lock Braking System | Prevents wheel lockup under braking |
| ASC | Automatic Stability Control | Traction control |
| CAN-bus | Controller Area Network | Vehicle communication network |
| CCV | Crankcase Ventilation | PCV system for blow-by gases |
| CDV | Clutch Delay Valve | Hydraulic clutch engagement restrictor |
| DISA | Differentiated Intake System | Variable intake manifold flap |
| DME | Digital Motor Electronics | Engine control unit (ECU) |
| DSC | Dynamic Stability Control | Electronic stability program |
| ETK | Elektronischer Teilekatalog | BMW electronic parts catalog |
| EWS | Elektronische Wegfahrsperre | Electronic immobilizer system |
| GM | General Module | Body electronics controller |
| IKE | Instrument Cluster Electronics | Instrument panel controller |
| IHKA | Integrierte Heizung Klima Automatik | Automatic climate control |
| LCM | Light Control Module | Exterior lighting controller |
| LSD | Limited Slip Differential | Traction-enhancing differential |
| MID | Multi-Information Display | Center console info screen |
| RTAB | Rear Trailing Arm Bushing | Rear suspension component |
| TIS | Technical Information System | BMW factory repair information |
| VANOS | Variable Nockenwellensteuerung | Variable valve timing system |
| ZHP | Zentrale Handling Paket | Performance package (N. America) |
