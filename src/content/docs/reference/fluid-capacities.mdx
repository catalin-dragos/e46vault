# Fluid Capacities

Complete fluid reference for the BMW E46 330 models. Use only the specified fluid types — incorrect fluids can cause damage.

## Critical Fluid Notes

- **Engine oil**: Must be BMW LL-01 approved
- **Coolant**: Blue BMW coolant only, 50/50 with distilled water
- **Power steering**: ATF Dexron III (most USA-market) or Pentosin CHF 11S (some European-market) — check fluid color before topping off (red = ATF, green = CHF 11S). Never mix the two types.
- **Transmission**: MTF-LT-2 for manual, Esso LT 71141 for automatic ZF
- **Brake fluid**: DOT 4, replace every 2 years regardless of mileage
