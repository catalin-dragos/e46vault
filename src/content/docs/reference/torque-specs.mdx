# Torque Specifications

Complete torque reference for the BMW E46 330. All values in Nm with ft-lb equivalent. Always use a calibrated torque wrench.

> **Note**: "Plus angle torque" means torque to the initial value, then rotate the additional degrees specified. Use a torque angle gauge for accuracy.

Refer to the interactive torque specification table below for a searchable reference of all values.

## Critical Fasteners

| Fastener | Torque | Notes |
|----------|--------|-------|
| Wheel lug bolts | 120 Nm (88 ft-lb) | Star pattern, re-torque after 100 km |
| Crankshaft pulley bolt | 410 Nm (302 ft-lb) | Requires special holder |
| Front wheel bearing nut | 290 Nm (214 ft-lb) | Replace nut |
| Cylinder head bolts | 40 Nm + 90° + 90° | Three-step angle torque |

## Important Reminders

- Always use new self-locking nuts where specified
- Apply threadlocker where specified (brake caliper carriers)
- "Torque at ride height" means tighten with suspension loaded (car on ground)
- Aluminum threads require extra care — torque values are lower
