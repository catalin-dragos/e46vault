# YouTube Channels

## BMW E46 Specific
- **50sKid**: Extremely detailed E46 DIY videos, covers nearly every repair
- **ShopLifeTV**: E46 maintenance and repair guides
- **Bavarian Built**: E46 build series and how-tos

## General BMW DIY
- **FCP Euro**: Professional-quality BMW repair videos
- **ECS Tuning**: Install guides for parts they sell
- **Humble Mechanic**: BMW/VW focused, good explanations
- **Deutsche Auto Parts**: BMW parts and install guides

## BMW Performance / Builds
- **Adam LZ**: E46 drift builds
- **jimmyoakes**: BMW build content
- **Speed Academy**: Performance builds including BMW

## Tips for Using YouTube Guides
- Verify torque specs against BMW TIS (not all YouTubers are accurate)
- Watch the entire video before starting the job
- Multiple videos on the same job give different perspectives
- Comments often contain helpful corrections and tips
