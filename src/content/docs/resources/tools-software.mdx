# Diagnostic Tools & Software

## BMW-Specific Software

### INPA / ISTA
- **INPA**: Legacy BMW diagnostic software, comprehensive module access
- **ISTA** (Rheingold): Current BMW dealer-level diagnostic tool
- Requires: K+DCAN USB cable (~$20-$30 on Amazon)
- Provides: Full fault code reading, actuator tests, coding, programming

### BimmerLink / BimmerCode
- Mobile app for iOS and Android
- Requires: Vgate iCar2 Bluetooth adapter (~$25)
- **BimmerLink**: Real-time data, fault codes, sensor readings
- **BimmerCode**: Module coding (enable/disable features)

## General OBD-II Tools
- **BlueDriver**: Quality Bluetooth OBD-II scanner, good for basic codes
- **Foxwell NT510 Elite**: Dedicated BMW scanner, affordable
- **Autel MaxiCOM**: Professional-grade, multi-brand

## What Each Tool Level Gets You

| Feature | Basic OBD-II | BimmerLink | INPA/ISTA |
|---------|-------------|------------|-----------|
| Read/clear DTCs | Yes | Yes | Yes |
| Live sensor data | Limited | Full | Full |
| BMW-specific codes | No | Yes | Yes |
| Module coding | No | Yes | Yes |
| Adaptation reset | No | Limited | Yes |
| Programming | No | No | Yes |
| Cost | $20-$50 | $25 + app | $25 cable + free software |
