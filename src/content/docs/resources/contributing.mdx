# Contributing to E46 Vault

E46 Vault is a community-driven knowledge base. We welcome contributions from all E46 enthusiasts.

## How to Contribute

### Report Issues
Found incorrect information? Use the Content Correction issue template on GitHub to let us know.

### Suggest Content
Have knowledge that should be documented? Use the New Content issue template to suggest topics.

### Submit Part Numbers
Have updated or superseded part numbers? Use the Part Number Update template.

### Submit Changes
1. Fork the repository on GitHub
2. Create a branch for your changes
3. Follow the content guidelines in CONTRIBUTING.md
4. Submit a Pull Request

## Content Standards

- All technical information must be verifiable
- Cite sources (BMW TIS, ETK, Bentley manual, etc.)
- Part numbers must be validated against BMW ETK or RealOEM
- Torque specs must reference official BMW TIS data
- Write for DIY mechanics of all skill levels

## Code of Conduct

Be respectful, accurate, and helpful. We're all here because we love these cars.
