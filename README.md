# E46 Vault

The encyclopedic reference for the BMW E46 330i, 330Ci, and 330xi (1999-2006).

**[Visit the site](https://catatwo.github.io/e46vault/)**

## What's Inside

250+ pages covering everything about the E46 330:

- Engine (M54B30), cooling, VANOS, intake, exhaust, ignition, fuel system
- Transmission (manual & automatic), clutch, driveshaft, differential
- Suspension, steering, brakes, wheels & tires
- Electrical systems, wiring, modules, lighting, audio
- Body, interior, HVAC
- Common problems & diagnosis
- Performance modifications
- Reference tables (torque specs, fluid capacities, part numbers, fault codes, fuse charts)
- Buying guide & resources

## Contributing

We welcome contributions from all E46 enthusiasts. There are several ways to help:

- **Quick edit** — Every page has an "Edit this page" link that takes you straight to the source file on GitHub
- **Report a correction** — [Open a Content Correction issue](https://github.com/catatwo/e46vault/issues/new?template=content-correction.yml) if you spot incorrect information
- **Suggest new content** — [Open a New Content issue](https://github.com/catatwo/e46vault/issues/new?template=new-content.yml) for topics that should be covered
- **Submit part numbers** — [Open a Part Number issue](https://github.com/catatwo/e46vault/issues/new?template=part-number.yml) for new or superseded part numbers
- **Submit a PR** — Fork the repo, make your changes, and open a pull request

See [CONTRIBUTING.md](.github/CONTRIBUTING.md) for content guidelines, style guide, and frontmatter requirements.

## Local Development

